# 전략 파라미터 가이드

스윙 눌림목 전략의 각 파라미터 값과 근거를 설명합니다.  
백테스트 데이터: 2021~2024년 KOSPI·KOSDAQ 전 종목, **5,015건**.

---

## TP / SL

### TP = +7% (tp_pct)

백테스트 전체 TP 청산 1,129건 분석 결과:

| 구간 | 건수 | 비중 |
|------|------|------|
| TP avg | +6.08% | — |
| TP max | +7.46% | — |
| +8% 이상 | 0건 | 0% |

**TP=13%로 설정 시**: TP 발동이 거의 없어 PF 0.566 → 0.039 붕괴.  
**TP=7%**: 실제 분포 상한에 맞춰 TP가 정상 작동. PF 최대.

> TP1 분할 익절 (`tp1_pct=0`)은 **비활성화**. TP=7%에서 절반만 수익 실현 시  
> 평균 winner 수익이 절반으로 줄어 PF 추가 하락.

---

### SL = bo_open × 0.99 (sl_buffer)

기준봉 시가(bo_open)의 99% 수준을 손절가로 설정.  
기준봉 시가 = 기술적 지지 구간으로, 이 아래 이탈 시 패턴 무효.

### SL 거리 한도 = 4% (sl_limit)

**배경**: 백테스트에서 SL -4~-5% 구간이 22% 존재.  
진입가가 bo_open보다 높을수록 (눌림 조금 더 진행된 뒤 진입) SL 거리 확대.

**효과**:
- sl_limit 10% → 4% 변경
- 평균 SL: -3.33% → -3.0% 개선 예상
- R:R: 1.83:1 → **2.03:1**
- 손익분기 승률: 35.4% → **33.0%**

---

### 하드스탑 = -5% (hard_stop_pct)

장 중 모니터링 공백(10:00→13:00)에서 급락 발생 시 최대 손실 한도.  
이전 7% → 5%로 축소한 이유:
- 09:10 갭오픈 SL 체크가 추가되어 갭 손실 대부분 조기 차단
- 7%는 R:R=0.87:1 (손익분기 53.5%) — 너무 관대
- 5%는 R:R=1.22:1 (손익분기 45.1%)

## 트레일링 스탑

| 파라미터 | 값 | 설명 |
|---------|-----|------|
| `trail_pct` | 5% | HWM 대비 -5%에 SL 설정 |
| `trail_activate_pct` | 3% | +3% 수익 이후에만 트레일링 개시 |

**activate_pct 도입 이유**:  
이전에는 `hwm = entry`로 초기화되어 `trail_sl = entry × 0.95`가 즉시 활성화.  
사실상 고정 5% SL로 작동 → 실전 TRAIL_SL 6건 전건 손실 (-4.97% 평균).  
+3% 수익 발생 후에만 트레일링을 개시해 수익 보호 역할 복원.

## 눌림목 조건

| 파라미터 | 값 | 설명 |
|---------|-----|------|
| `pullback_vol` | 0.7배 | 기준봉 이전 20일 평균 거래량의 70% 이하 |
| `pullback_shape` | 20% | 캔들 몸통 / 전체 범위 ≤ 20% (도지형) |

거래량 감소 + 도지형 캔들 = 매도세 소진, 눌림목 완성 신호.

## 시장 필터

```
get_kospi_condition() 통과 조건:
  1. KOSPI 종가 ≥ MA20 (현재가가 중기 추세 위)
  2. MA20 now ≥ MA20 5거래일 전 (MA20 기울기 상승)
  3. 5거래일 수익률 > -3% (급락 브레이크)
```

조건 3 (주간 급락 브레이크)은 백테스트 최대 연속 손실 22회 중  
상당 부분이 지수 급락 구간에서 발생한 점에서 추가.

## 포지션 리스크 관리

| 파라미터 | 값 | 설명 |
|---------|-----|------|
| `max_positions` | 5 | 최대 동시 보유 종목 |
| `max_sector_count` | 2 | 동일 섹터 최대 2종목 (집중 리스크 차단) |
| `max_hold_days` | 7 | 기간 만료 기준 (EXPIRE 평균 7.0일) |

**신호점수 비례 사이징**:

| 신호점수 | 사이징 배율 | 실투자금 (기준 100만원) |
|---------|------------|----------------------|
| 40~59점 | × 0.6 | 60만원 |
| 60~79점 | × 1.0 | 100만원 |
| 80점↑   | × 1.5 | 150만원 |

> 미검증 팩터이므로 ±50% 범위로 제한. 신호점수와 미래 수익의 상관관계 확인 후  
> 범위 확장 검토.

## 2차 검증 조건 (15:20)

KIS API 실시간 데이터로 1차 후보를 재검증:

| 조건 | 기준 |
|------|------|
| 거래량 | ≤ 전일 20일 평균 × 0.9 |
| 지지 | 현재가 ≥ bo_open |
| 캔들 형태 | (cur - open) / range ≤ 30% |
| 체결강도 | ≥ 110 |
| 신호점수 | ≥ 40점 |
| SL 거리 | ≤ 4% |

## 드리프트 감지

3주 연속 주간 승률 < 35% 시 텔레그램 경고 발송.  
(손익분기 33%보다 2%p 높게 설정 — 통계적 노이즈 구분)

파라미터 재검토 권장 트리거:
1. 전략 드리프트 경고 3회 이상
2. 월별 PF < 0.4
3. 최대 연속 손실 ≥ 10회
